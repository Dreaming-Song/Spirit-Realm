# 游戏设计文档

## 世界观
_待补充_

## 核心玩法循环

1. **探索** → 在仙侠世界中自由探索（御剑飞行、徒步）
2. **采集** → 采集草药、矿石等物理材质
3. **炼丹** → 利用采集的材质炼丹，调整火候影响丹药效果
4. **成长** → 丹药增强能力，解锁新法术/灵宠技能
5. **挑战** → 探索宗门秘境，组队解谜，战斗妖兽

## 核心系统

### 御剑飞行
- 物理驱动：按气流调整高度
- 碰撞树木/障碍物会坠机
- 顺风加速、逆风减速

### 五行法术
- **金**：破解机关锁、金属操控
- **木**：催生树木、藤蔓生长
- **水**：冰冻、治疗
- **火**：点燃草丛、熔炼
- **土**：召唤巨石推障、地形改变

### 炼丹系统
- 采集草药/矿石等物理材质
- 放入丹炉，调整火候/时间
- 不同材质组合 → 不同丹药
- 配方存储于数据库

### 灵宠系统
- 仙鹤、灵狐等
- 喂食对应食材解锁技能
- 载人飞行、辅助战斗

### 宗门秘境
- 物理解谜
- 用法术调整机关
- 支持联机组队协作
