## 🎯 核心玩法循环

```
探索世界 → 采集材料 → 炼丹/育宠 → 变强 → 挑战秘境 → 🌀
  ↑                                                        |
  └──────────────────────── 联机组队 ───────────────────────┘
```

1. **探索** → 御剑/徒步在仙侠世界自由探索
2. **采集** → 草药、矿石、灵宠食材等物理拾取
3. **炼丹/育宠** → 采集材料炼丹；喂食灵宠解锁技能
4. **成长** → 丹药增强属性，灵宠进化/学新技能
5. **挑战** → 宗门秘境解谜、妖兽战斗
6. **联机** → 2-4人组队，共享探索和战斗

## ⚔️ 核心系统

### 御剑飞行
- 物理驱动：按气流调整高度
- 碰撞树木/障碍物会坠机
- 顺风加速、逆风减速

### 五行法术
- **金**：破解机关锁、金属操控
- **木**：催生树木、藤蔓生长
- **水**：冰冻、治疗
- **火**：点燃草丛、熔炼
- **土**：召唤巨石推障、地形改变

### 炼丹系统
- 采集草药/矿石等物理材质
- 放入丹炉，调整火候/时间
- 不同材质组合 → 不同丹药
- 配方存储于数据库

### 灵宠系统（核心！参考洛克王国）
- 种类：仙鹤、灵狐、竹熊、小貔貅等 Q 萌灵宠
- 获取：野外遭遇 → 战斗/喂食 → 收服
- 养成：喂食 → 升级/进化 → 解锁技能
- 功能：载人飞行、辅助战斗、采集助手
- 联机：可展示灵宠、组队携带多宠

### 宗门秘境
- 物理解谜（推石、点火、机关）
- 法术与环境交互
- 支持联机组队协作
- 通关奖励：稀有丹药配方、灵宠蛋
